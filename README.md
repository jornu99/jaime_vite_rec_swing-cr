Aplicación web para la planificación de horarios y gestión de espacios del festival de Swing. Permite la creación, visualización y organización de eventos mediante un sistema dinámico.

## 🛠️ Tecnologías
* **Frontend**: JavaScript Vanilla
* **Herramientas**: Vite
* **Persistencia**: LocalStorage

## 💻 Ejecución
Para visualizar el proyecto en local, ejecute:

1. **Instalar dependencias:**
   npm install

2. **Arrancar servidor de desarrollo:**
   npm run dev

El proyecto se desplegará en la URL indicada por la terminal (usualmente http://localhost:5173).
